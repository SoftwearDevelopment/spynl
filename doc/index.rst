Welcome to Spynl's documentation!
=================================

Spynl is a Python web framework built on top of  `Pyramid <http://getpyramid.com>`_.

It comes with some opinionated included batteries for modern web development
like translations, multiple content types, HTML emails and building with Docker and Jenkins.

Quickly get to writing your actual features in a Spynl plugin and get out there.
